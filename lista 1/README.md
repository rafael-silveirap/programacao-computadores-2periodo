## Exercício 1:
Escreva um programa que leia um número inteiro positivo do teclado e informe se ele é múltiplo de 3. <br>
Exemplos:
```
Digite um numero: 9
Esse número é multiplo de três!
```
```
Digite um numero: 17
Esse número não é multiplo de três!
```
